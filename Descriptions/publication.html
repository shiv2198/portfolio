Visual Appearance Based Person Retrieval In Unconstrained Environment Videos

Project information

  • Project date: September, 2019
  • Publication Name: Image and Vision Computing, Elsevier, 2019
  • Paper URL: Paper

This is a research paper on the field of deep learning and computer vision. The aim of the paper is building a model to detect person in an unconstrained environment video with only soft-biometric description of the person. For the image segmentation part of the mode Mask-RCNN is used and for the classification of biometric trait densenet is trained for each trait.
